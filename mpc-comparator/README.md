Проект состоит из основных четырёх модулей:
- `beaver_triple_generator.py` — генерация входных данных и троек Бивера;
- `check_generator.py` — проверка корректности генерации и восстановления чисел;
- `party.py` — реализация логики MPC-сравнения для каждой стороны;
- `utils.py` — вспомогательные функции для безопасных бинарных операций.

Для запуска:
1. Остановить и удалить прерыдущие контейнеры: 
```bash
docker compose down
```

2. Запустить, пересобрав новые образы:
```bash
docker compose up --build
