# Генерация мультипликативных троек Бивера

Логика генерации троек Бивера реализована в файле `party.py`.  
Вспомогательные функции для отправки и приема данных находятся в файле `utils.py`.

Сгенерированные тройки сохраняются в файлы `p1.csv` и `p2.csv` в папке `shared`.  
Файл для проверки корректности троек — `mul.py`, который также находится в этой папке.  

> Важно: `mul.py` использует относительные пути к файлам, поэтому код необходимо запускать из папки `shared`.  
> Пример запуска:
> ```bash
> python3 mul.py

Для генерации троек используйте команду:
```bash
docker compose up --build
```
